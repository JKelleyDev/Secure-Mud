# Quests — given by NPCs in the tavern, completed by turn-in.
#
# Format: "[quest: <id>]" then fields.
#   name         — display name
#   giver        — mob id of the NPC who gives this quest
#   needs        — items required at turn-in: "item_id=count, ..."
#   reward_gold  — gold paid on turn-in
#   reward_xp    — xp granted on turn-in
#   reward_item  — single item id granted on turn-in (optional)
#   next         — id of the quest that unlocks after this (optional)
#   storyline    — phase id to flip the world into on turn-in (optional;
#                  must match an id in storyline.txt)
#   desc         — quest text the NPC says when offering it
#
# Important: the offer order is currently hardcoded in
# server/engine.py:_current_quest_offer (the "chain" list). If you add a
# brand new chain of quests, that list needs updating too. See README.
